Bienvenido a la Gestión de Productos

Acerca del Proyecto

Esta es una aplicación básica para la gestión de productos.
Utiliza Spring Boot en el backend y PostgreSQL como base de datos.
Puedes agregar, ver, actualizar y eliminar productos desde la API.

¿Cómo empezar?

  • Configura la base de datos en application.properties.
  • Ejecuta el backend para exponer la API REST.
  • Accede a los endpoints para gestionar productos.

Para más información, consulta la documentación del proyecto.
